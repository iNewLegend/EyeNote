# EyeNote - Business Models

## Core Business Models

### 1. Freemium SaaS for Teams
**Overview:**
- **Free Tier:** Basic annotation on web pages and IDEs with limited groups
- **Pro Tier ($10/user/month):** Unlimited groups, advanced collaboration, and priority support
- **Enterprise Tier (Custom pricing):** SSO, advanced security, and dedicated infrastructure

**Target Market:**
- Web development teams
- Software engineering teams
- Product management teams
- Remote development workforces

### 2. Enterprise Licensing
**Features:**
- Self-hosted deployment options
- Custom IDE plugin development
- Advanced analytics on code annotation
- Dedicated account management

**Pricing:**
- Annual subscription based on team size
- Volume discounts for large organizations
- Custom development services available

## Industry-Specific Solutions

### 1. Web Development
- **Features:** Browser-based annotation, code element targeting, team collaboration
- **Targets:** Web agencies, frontend teams, full-stack developers
- **Concrete Use Cases:**
  - Developers leaving contextual feedback directly on UI elements during code review
  - QA teams reporting bugs with precise element references and console logs
  - Product managers collecting feedback on new features from stakeholders
  - Designers providing visual feedback on implemented designs
  - Remote teams collaborating on website content updates

### 2. Software Development
- **Features:** IDE integration, code annotation, pull request integration
- **Targets:** Engineering teams, open-source projects, tech startups
- **Concrete Use Cases:**
  - Code reviews with inline comments tied to specific code elements
  - Onboarding new developers with contextual codebase explanations
  - Documenting technical debt and future improvements
  - Cross-team collaboration on shared codebases
  - Maintaining living documentation within the code

### 3. Digital Agencies
- **Features:** Client feedback collection, versioned annotations, approval workflows
- **Targets:** Web design agencies, marketing agencies, digital studios
- **Concrete Use Cases:**
  - Collecting client feedback directly on website mockups
  - Tracking design implementation progress across team members
  - Maintaining version history of client feedback iterations
  - Streamlining approval processes with visual annotations
  - Collaborating with external contractors on specific elements

### 4. E-learning & Technical Training
- **Features:** Interactive tutorials, code walkthroughs, student feedback
- **Targets:** Coding bootcamps, online course creators, technical educators
- **Concrete Use Cases:**
  - Creating interactive coding exercises with inline hints
  - Providing personalized code reviews to students
  - Building guided tours of complex web applications
  - Collecting student questions on specific UI elements
  - Creating annotated code examples with explanations

### 5. Technical Documentation
- **Features:** Contextual help, interactive guides, feedback collection
- **Targets:** Developer relations, technical writers, API teams
- **Concrete Use Cases:**
  - Creating interactive API documentation with live examples
  - Collecting feedback on documentation accuracy and clarity
  - Building contextual help systems for web applications
  - Annotating complex workflows within web interfaces
  - Maintaining up-to-date onboarding materials
